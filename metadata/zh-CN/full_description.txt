* 能运行人生游戏化、大纲、人生原则整理等等，支持太微平台上 800+ 插件的笔记应用。
* 本地优先、使用 tid/md 纯文本文件且支持富文本编辑，与太记桌面版免费同步。
* 建议先安装太记桌面版，然后扫码同步到移动端。在桌面端创建的知识库里，侧边栏的「MobileSync」有二维码可以扫。
* 当然也可以只使用移动端，直接点击「添加工作区」，可以通过几个在线模板创建手机版的新知识库
* 可以直接点击 + 按钮创建笔记，简单地开始使用
* 高级用法非常丰富，参考自带的「帮助手册」可以看教程。
* 有许多可供选择的布局，基本上就像知识库中的小程序。你可以通过一个看起来像品字形的按钮打开它们。
* 使用 CPL 布局，你可以搜索并安装 800 多个免费开源插件，并阅读它们的文档网站。
* 你可以直接在手机上使用 HTML CSS 和 JS 或 WikiText 开发自己的小程序，并通过 CPL 安装的 Gatha 插件打包并发布给其他人使用。或使用这些技术完全地定制化自己的知识库。别忘了定期往桌面端备份数据。
